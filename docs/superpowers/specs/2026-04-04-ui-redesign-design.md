# UI 리디자인 설계 — 좋은 질문 연습실

## 개요

Streamlit 기반 "좋은 질문 연습실" 앱의 전체 UI를 다크 테마 기반으로 리디자인한다.
코칭 결과를 점수 카드 대시보드 + 접히는 상세 패널 방식으로 변경하고,
퀴즈/사례 검색 화면도 카드 기반 레이아웃으로 통일한다.

## 디자인 결정 사항

### 1. 전체 테마

- **다크 모드** (전문적 & 세련된 스타일)
- 배경: `#0D1117`, 사이드바: `#161B22`, 카드: `#1C2128`
- 보더: `#21262D`, 텍스트: `#E5E7EB`, 보조 텍스트: `#7D8590` / `#8B949E`
- Streamlit `config.toml` 테마 설정으로 기본 색상 적용 + `st.markdown` CSS 오버라이드

### 2. 색상 팔레트

| 용도 | 색상 | 코드 |
|------|------|------|
| 배경 | 다크 | `#0D1117` |
| 카드 배경 | 다크 그레이 | `#1C2128` |
| 보더 | 미디엄 그레이 | `#21262D` |
| 프라이머리 (버튼, 액센트) | 블루 | `#1F6FEB` / `#58A6FF` |
| 경고/점수 | 골드 | `#F59E0B` |
| 성공/개선 | 그린 | `#3FB950` |
| 위험/개선필요 | 레드 | `#DA3633` / `#EF4444` |
| 텍스트 | 라이트 | `#E5E7EB` |
| 보조 텍스트 | 그레이 | `#8B949E` |

### 3. 코칭 모드 — 결과 대시보드

현재: 하나의 마크다운 텍스트 블록으로 모든 결과 출력
변경: **점수 카드 3개 + 접히는 상세 패널(expander) + 액션 버튼**

#### 점수 카드 (상단 3열 그리드)
- **질문 점수**: 큰 숫자 (골드 색상) + `/10` 서브텍스트
- **문제 유형**: 텍스트 (블루 색상)
- **점수 변화**: `+N` / `-N` (그린/레드)

#### 접히는 상세 패널 (Streamlit `st.expander` 활용)
1. **진단 상세 보기** — 배지: "개선 필요" (레드)
2. **개선된 질문 보기** — 배지: "리라이팅" (블루), 체크마크로 개선 포인트 표시
3. **예시 답변 비교** — 배지: "Before / After" (그린), 2열 비교 레이아웃

#### 액션 버튼 (3열 그리드)
- **만족해요** (프라이머리 블루)
- **직접 수정할게요** (세컨더리 그레이)
- **다시 분석해줘** (세컨더리 그레이)

### 4. 사이드바

- 모드 선택: 라디오 버튼 → 아이콘 + 텍스트 버튼 스타일 (활성 모드 하이라이트)
- 질문 맥락: 다크 테마 입력 필드
- 병렬 처리: 토글 스위치
- 점수 변화 그래프: **모노톤 블루 바 차트** (`#30363D` → `#58A6FF` 그라데이션 + 은은한 glow)
- 하단: 세션 저장 (프라이머리) / 새로고침 버튼

### 5. 퀴즈 모드

#### 퀴즈 출제 카드
- 헤더: `QUIZ` 배지 (레드) + "이 질문의 문제점은?"
- 본문: 인용 박스 스타일로 나쁜 질문 표시
- 힌트: 블루 배경 박스
- 답변 입력 + 제출 버튼

#### 퀴즈 결과 카드
- 헤더: 정답/오답 배지 (그린/레드) + "평가 결과"
- 평가 내용 박스
- 좋은 질문 예시: 그린 테두리 박스
- "다음 퀴즈 풀기" 버튼

#### 사이드바 추가
- 퀴즈 기록: 정답/오답/정답률 통계 표시

### 6. 사례 검색 모드

#### 검색 결과 카드
- 헤더: "사례 검색" 배지 + 주제 타이틀
- 좋은 질문 사례: 번호 배지 + 질문 텍스트 리스트
- 추천 프레임워크: 이름/설명 + 단계별 태그 (예: STAR)

#### 사이드바 추가
- 최근 검색 기록 리스트

### 7. 공통 요소

- 유저 메시지: 블루 버블, 우측 정렬
- 채팅 입력: 다크 테마 입력 필드 + 전송 버튼
- 페이지 타이틀: `💡 좋은 질문 연습실`

## 구현 방식

### Streamlit CSS 커스터마이징

`st.markdown`의 `unsafe_allow_html=True`로 커스텀 CSS 주입:
- `.stApp` 배경색 오버라이드
- `.stSidebar` 스타일 오버라이드
- `st.expander` 스타일 커스터마이징
- `st.columns` + `st.metric` 조합으로 점수 카드 구현
- `st.button` 스타일 오버라이드

### config.toml 변경

```toml
[theme]
primaryColor = "#58A6FF"
backgroundColor = "#0D1117"
secondaryBackgroundColor = "#161B22"
textColor = "#E5E7EB"
```

### 파일 변경 범위

- `streamlit_app.py` — UI 레이아웃 전면 변경
- `.streamlit/config.toml` — 테마 색상 변경

### 기능 변경 없음

- LangGraph 워크플로우, 노드, 에이전트, 프롬프트 등 백엔드 로직 변경 없음
- 상태 관리 구조 유지
- 기존 기능(코칭/퀴즈/사례 검색/세션 저장/내보내기) 모두 유지
